# 🚀 TaskFlow - Strategic Task Manager

**TaskFlow** is not just a to-do list; it's a demonstration of modern frontend fundamentals. It bridges the gap between simple functionality and professional user experience.

## 🧠 Core Competencies Demonstrated
- **State Persistence:** Implemented `LocalStorage` to ensure data remains intact across browser sessions.
- **Dynamic UI Updates:** Real-time task counters (Pending/Completed) using DOM manipulation.
- **Advanced Logic:** A priority-based sorting system (High/Medium/Low) for better productivity.
- **Theming:** A full Dark Mode implementation using CSS Variables and JS state tracking.

## 🛠️ Problem-Solving Journey
During development, I encountered and resolved several technical hurdles:
1. **The 'Enter' Key UX:** Standardized user input by mapping the Enter key to the addition function, improving speed.
2. **Component Conflicts:** Resolved ID duplication issues that occurred during the layout expansion, ensuring a clean and bug-free DOM structure.
3. **Responsive Scaling:** Engineered a CSS Grid/Flexbox hybrid layout to maintain usability on mobile devices.

## 📂 Project Structure
- `index.html`: Semantic structure and accessibility.
- `style.css`: Custom styling with a focus on Glassmorphism and responsive design.
- `script.js`: Clean, modular JavaScript logic.
